## 2. Описание Эффекта
**Название**: Линейное Разворачивание (Linear Unfold) / Раскрытие Масштабом (Scale Reveal).

**Визуальное описание**: 
Изображение появляется не через физический изгиб бумаги, а скорее через эффект "растягивания" или "выезжания" из сложенного состояния (возможно, масштаб по одной оси меняется от 0 до 1, либо используется маска).

**Математическая модель**:
$$ P_{final}(x, y) = P_{folded}(x, s_y \cdot y + t_y) $$
Где $s_y$ (scale) меняется от ~0 до 1, либо аналогично для оси X. Скорее всего, это **Affine Transformation** (Non-uniform scaling) или **Linear Mask Reveal**.

## 3. Вывод для Анимации
Предыдущие попытки использовать модификатор `Simple Deform (Bend)` были **избыточны** и создавали лишнюю геометрию (дугу), которой нет в референсе.
**Рекомендация**: Использовать простую анимацию масштаба (Scale) по одной из осей (вероятно Y, судя по предыдущему анализу активных строк), либо простую плоскую анимацию "жалюзи".
